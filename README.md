# Fan-RPM-Detection


A FAN Rotate Per Minute Detection Test Instrument


### 文档tree如下：
	.
	├── README.md
	├── Fan 风扇学习报告.xls

### 这个项目是关于Fan 风扇的学习报告 以及 一个利用光电传感器来测试风扇的每分钟转速的。


`项目使用的主芯片是AT89S52芯片，自带有3个Timer，我们主要是使用Timer来产生计时和计数功能，光电传感器来检测风扇叶片遮挡与否，想了解更多请参考文档，源码见文档中。`
